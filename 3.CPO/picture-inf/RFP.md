# RFP: 旅行写真管理・日本地図可視化システム

## 概要

旅行写真をフォルダ単位で管理し、写真のEXIF情報から訪問都道府県を自動抽出して、日本白地図に可視化するWebアプリケーション。

---

## 要件一覧

### 1. フォルダ・データ構成

- フォルダ名フォーマット：`YYYYMMDD_タイトル`（例：`20260422_東京旅行`）
- フォルダはブラウザからアップロード形式で取り込む
- アップロード時に以下を入力フォームで受け付ける
  - 旅行タイトル（フォルダ名から自動取得・編集可）
  - 旅行の説明文（テキスト入力）

---

### 2. 写真選択

- アップロードしたフォルダ内の写真を一覧表示
- ユーザーが手動で **5枚** を選択する
- 選択した5枚が旅行ページに表示される

---

### 3. 旅行ページ（各旅行ごとに1ページ）

- 表示内容
  - 旅行タイトル
  - 旅行の説明文
  - 選択した写真5枚（サムネイル表示）
  - 各写真のEXIF情報（撮影日時・撮影場所・GPS・撮影機材）
- 白地図ページへのリンクを設置

---

### 4. 都道府県の自動抽出

- フォルダ内 **全写真** のEXIF（GPS情報）からジオコーディングで都道府県を自動抽出
- 抽出した都道府県をデータベースに保存（旅行と都道府県の紐付け）
- 1つの旅行で複数都道府県にまたがる場合はすべて記録
- 最終目標：日本の47都道府県すべてを塗りつぶす

---

### 5. 白地図ページ

- 日本の都道府県別白地図をWebページとして表示
- 訪問済みの都道府県を塗りつぶし
  - 訪問回数が多いほど色が濃くなる（段階的な濃淡）
- 各旅行ページから白地図ページへリンク
- 白地図ページから各旅行ページへリンク

#### 都道府県クリック時の動作

- ポップアップが開く
- ポップアップ内に、その都道府県に行った旅行タイトルを **時系列順** で一覧表示
- タイトルをクリックすると該当の旅行ページへ遷移

## 技術スタック

| 領域 | 技術 |
|------|------|
| Backend | Python + Flask |
| Database | SQLite + SQLAlchemy |
| 地図 | D3.js + 日本GeoJSON |
| フロントエンド | Jinja2 + Bootstrap 5 + Vanilla JS |
| 画像処理 | exiftool + Pillow |
| ジオコーディング | Nominatim（OpenStreetMap） |

## 画面一覧

| 画面 | URL | 説明 |
|------|-----|------|
| TOP / アップロード | `/` | フォルダアップロード＋タイトル・説明文入力 |
| 写真選択 | `/trips/<id>/select` | アップロード済み写真から5枚選択 |
| 旅行ページ | `/trips/<id>` | 写真5枚＋EXIF情報＋説明文 |
| 旅行一覧 | `/trips` | 全旅行の一覧 |
| 白地図 | `/map` | 都道府県別日本地図・訪問状況可視化 |
